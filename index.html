Datahamstern

Strukturerad information om Sverige

Vad är detta?

En öppen databas, för närvarande innehållande alla svenska organisationer, myndigheter, län, kommuner, orter, postnummer, gator och husnummer. Målet är att tillhandahålla "all tänkbar" samhällsnyttig information om Sverige.

Varför?

Samhällsnyttig information om Sverige är svindyrt

Lantmäteriet och bolagsverket är två exempel på myndigheter som samlar information de tar väldigt mycket betalt för. Exempelvis kostar ett utdrag med polygoner över Sveriges kommuners gränser nnnn kronor, ett utdrag med Sveriges 30 000 bostadsrättsföreningars registreringsdatum kostar 18 000 kronor och vill man dessutom ha information om bostadsrättsföreningarnas fastigheter blir priset flera hundra tusen kronor.

Priserna är exklusive moms och med avtalad begränsning hur den får användas och publiceras.

Små aktörer stängs ute

Kostnaden för datautdrag är försvinnande liten om man är ett företag med tjock plånbok. De som saknar kapital, exempelvis de flesta småföretagare och privatpersoner, ges väldigt få möjligheter.

En lokal databas, inte bara ett API

Mycket av den information som tillhandahålls i databasen finns redan tillgängligt via öppna API:er. Det är dock inte alltid ett API är nog, exempelvis vid stora operationer kan det löna sig väldigt mycket att utan fördröjning ha informationen tillgänglig. Många gånger vill man dessutom strukturera om information i olika dimensioner för att kunna nå det man är ute efter. Ett API som hämtar en post via HTTP är då inte nog.

Data

Komplexitet

Funktioner som för närvarande anses vara väsentliga nog för slutanvändare är:

  1. Historik: vilket värde hade en post vid ett givet datum? När noterades det nuvarande värdet första eller senaste gången?
  2. Fullständiga källhänvisningar: i vilka källor har informationen hittats?
  3. Licenshantering: vem äger rättigheterna till informationen?
  4. Variabel tillförlitlighet: hur mycket kan man lita på att informationen stämmer?

Denna information finns tillgängligt för alla värden och poster i databasen. Exempelvis kan man ta reda på när ett företag sågs till första gången, när företagets namn senast ändrades och vilka källor som används för att nå fram till i vilket län företaget är registrerat.

API

Databasen finns tillgänglig via Java eller som JSON via HTTP. För de som inte vill eller inte kan installera och administrera en lokal databas så finns en publik webtjänst att tillgå.

Källor

Den lokala databasen är ett resultat från att ha konsoliderat information från flera källor, för närvarande:

  1. Wikipedia: län, kommuner, tätorter och småorter.
  2. Näringslivsregistret: registrerade organisationsnummer, firmanamn, firmaform, status och hemlän.
  3. Posten: postorter, postnummer, gator och husnummer.
  4. Fastighetsmäklarnämnden: detaljer om organisationer med koppling till fastighetsmäkleri.
  5. hydda.se: detaljer om bostadsrättsföreningar och relaterade dokument som ekonomiska planer, årsredovisningar och stadgar.
  6. Open street map: geografiska koordinater till gatuadresser, läns- och kommungränser, etc.
